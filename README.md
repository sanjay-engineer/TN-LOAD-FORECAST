# ⚡ TN Intelligent Load Forecasting System

A complete online web application for Tamil Nadu Power Grid 
day-ahead load forecasting using LSTM neural networks.

## Features
- User registration and login system
- Each user has private data space
- Upload Colab results (CSV + charts)
- Interactive dashboard with Plotly charts
- MAPE and RMSE accuracy tracking
- Rolling forecast — gets smarter every day
- Works on any device — phone or laptop

## How to Use
1. Register an account on the app
2. Run `TN_COMPLETE_COLAB.py` in Google Colab
3. Download results from Colab
4. Upload to this dashboard
5. See your predictions and accuracy

## Tech Stack
Python, Streamlit, Plotly, Pandas, TensorFlow (Colab)

## Live App
Deployed on Streamlit Cloud
